# PROJET : SAAS_TIKTOKSHOP
Ce projet est un SaaS B2C permettant de générer automatiquement des vidéos UGC TikTok réalistes à partir d'un lien d'un produit tiktok shopm, grâce à une orchestration d'IA.

## 🛠️ ARCHITECTURE TECHNIQUE VISÉE
* **Front-end & Logique :** Next.js (App Router), React, TypeScript.
* **Style & Design :** Tailwind CSS (utiliser des interfaces modernes, propres, typées SaaS B2C).
* **Backend & Base de données :** Supabase (Auth, Postgres, Storage).
* **Gestion du Stockage :** Les vidéos générées seront stockées sur Supabase mais DEVRONT inclure une logique de suppression automatique (TTL) après 7 jours pour économiser la bande passante.
* **Monétisation :** Whop (Webhooks pour gérer les formules d'abonnements et l'attribution des crédits).
* **Moteur IA Vidéo :** API Higgsfield (pour l'orchestration génération vidéo/image).
